## redux介绍
#### 官网
https://redux.js.org

#### Redux 是什么？
Redux is a predictable state container for JavaScript apps

#### Redux概念
store: 应用数据的存储中心
action: 应用数据的改变的描述
reducer: 决定应用数据新状态的函数，接收应用之前的状态和一个 action 返回数据的新状态。
state: 状态
middleware: redux 提供中间件的方式，完成一些 流程的自定义控制，同时形成其插件体系。
